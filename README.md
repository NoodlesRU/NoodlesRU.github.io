# Personal Portfolio Website
This ReadME was created using AI for ease of understanding.
A responsive portfolio website built with semantic HTML, CSS, and Tailwind CSS.

## Features

- **Semantic HTML**: Properly structured HTML5 with semantic elements
- **Responsive Design**: Mobile-first approach that works on all screen sizes
- **Tailwind CSS**: Utility-first CSS framework for rapid styling
- **Modern UI**: Clean, professional design with smooth animations
- **Accessible**: Built with accessibility best practices in mind

## Getting Started

### Prerequisites

- Node.js (for development)
- Python (for local server, optional)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/NoodlesRU/NoodlesRU.github.io.git
cd NoodlesRU.github.io
```

2. Install dependencies:
```bash
npm install
```

### Development

1. **View the website**: Simply open `index.html` in your browser, or use a local server:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`

2. **Using npm scripts**:
```bash
# Start local server and open in browser
npm start

# Just start the server
npm run serve
```

### Building CSS (Optional)

If you want to build Tailwind CSS locally:
```bash
npm run build
```

## Responsive Design

The website is built with a mobile-first approach:

- **Mobile (< 768px)**: Single column layout, hamburger menu
- **Tablet (768px - 1024px)**: Two-column grid for features
- **Desktop (> 1024px)**: Full three-column grid, horizontal navigation

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Internet Explorer 11+ (limited support)
